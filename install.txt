Installing AuditSurf for Alfresco 3.3E or higher version


1. Install the AMP

   a. Copy the AMP file into the "amps/" folder of your Alfresco.

   b. Run apply_amps.sh


2. Install the web application

   a. Copy the WAR file into "tomcat/webapps/"


3. You have to add a line to "alfresco-global.properties" file to enable audit :
   audit.enabled=true       in alfresco 3.4 or higher version
   audit.useNewConfig=true  in alfresco 3.3

Remarks :

- Compatible with Alfresco V3.3 or higher version ; JMX obviously needs an Enterprise release

- The AMP audits the following services:
   - FileFolderService
      * create
   - NodeService
      * createNode
   - CheckoutCheckinService
      * checkin
   - ContentService
      * getReader
   - AuthenticationService
      * authenticate
   - PersonService
      * createPerson
   - WorkflowService
      * startWorkflow
